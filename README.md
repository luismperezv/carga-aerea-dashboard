# Carga Aerea Dashboard

A modern, interactive analytics dashboard for monitoring air cargo operations.

## Features

- **Hero Stats**: High-level overview of total parcels, inspection rates, and missing data alerts.
- **Dynamic Comparison**: Side-by-side comparison columns allowing users to toggle between historical data and monthly snapshots (e.g., January 2022).
- **Interactive Charts**:
    - **Monthly Trends**: Global volume trends across different cargo labels.
    - **Label Distribution**: Doughnut charts for granular cargo type analysis using Chart.js.
- **Modern UI**: Built with a custom design system focusing on readability and professional aesthetics.

## Tech Stack

- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- [Chart.js](https://www.chartjs.org/) for data visualization

## How to Run

Simply open `index.html` in any modern web browser.
